# Digital-Banking-

Overview
This is my solution to the Easybank Landing Page challenge from Frontend Mentor. The goal was to build a clean, responsive landing page using only HTML and CSS. I focused on getting the layout, spacing, and general structure in place.

What I Built

A hero section with text and mockup images

A features section using a simple grid

A latest articles section

A footer with social icons and links

Everything is styled with basic Flexbox and Grid.

Tools Used
HTML5

CSS3 (Flexbox + Grid)

Frontend Mentor starter files and design assets

Notes
I didn’t include JavaScript for the mobile menu — instead, I used a simple checkbox toggle to keep things lightweight. My main focus was getting a functional layout that matches the design as closely as possible within the time I had.
